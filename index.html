📋 Общий обзор
🎯 Концепция игры

Игрок получает угрожающее сообщение от незнакомца, который утверждает что нашёл его IP. Нужно использовать OSINT-техники чтобы вычислить незнакомца первым, собрать доказательства и подать анонимный донос в полицию.

🕹️ Жанр и стиль
  • FPS от первого лица — ходим по комнате
  • Симулятор хакера с реальными техниками
  • Атмосфера Kali Linux / киберпанк
  • Нарративный детектив с OSINT механиками

🎮 Управление в комнате
  • WASD — передвижение
  • Мышь — обзор камеры
  • E — взаимодействие с объектами
  • ESC — пауза / выход из ноутбука
  • F — фонарик (опционально)
✨ Атмосфера
  • Ночное время суток
  • Синяя/фиолетовая подсветка мониторов
  • Звуки вентилятора ПК, клавиатуры
  • Уведомление на телефоне в начале игры
  • Постеры на стенах (хакерская тематика)
🎮 Геймплей
📖 История

Незнакомец пишет тебе угрожающее сообщение и утверждает что знает твой IP. Ты должен вычислить его первым — найти личность, собрать досье и сдать в полицию.

🔍 OSINT механики
  • Поиск по нику в соцсетях
  • Анализ IP адреса через терминал
  • Сбор информации через nmap, whois, dig
  • Просмотр профилей в браузере
  • Составление досье на цель
  • Анонимный донос в полицию
🖥️ Интерфейс Kali OS
  • Рабочий стол с иконками
  • Терминал с реальными командами
  • Браузер для OSINT поиска
  • Мессенджер для общения с целью
  • Файловый менеджер для досье
🎬 Катсцены
  • Вступление — получение угрозы
  • Обнаружение цели — момент "нашёл"
  • Финал — отправка доноса в полицию
